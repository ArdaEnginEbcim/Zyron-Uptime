# Zyron-Uptime

## 👍 Start Bot
- Edit line 6 in the main file
```js
client.login("TOKEN");
```

## 📺 Main File
- `./zyronup.js`

## 🌐 Debug with VSCode (Windows)
* `F5` -> start
* `Shift + F5` -> stop
* `Ctrl + Shift + B` -> run build task
* `F9` -> toggle breakpoint
* `F10` -> step over
* `F11` -> step into
* `Shift + F11` -> step out
